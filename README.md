## ✨ 功能特点 | Features

- 🖼️ **智能扫描** - 自动识别当前网页的所有图片
- ✅ **批量选择** - 支持单选、全选、反选操作
- 📏 **尺寸过滤** - 一键筛选大图（≥500x500像素）
- 💾 **批量下载** - 支持跨域图片下载
- 🎨 **美观界面** - 现代化设计，操作直观流畅
- ⚡ **性能优异** - 轻量级，不影响浏览速度

## 📦 安装方法 | Installation

### Chrome Web Store（推荐）
即将上架到 Chrome 应用商店...

### 手动安装 | Manual Installation

1. 下载或克隆此仓库
   ```bash
   git clone https://github.com/airobus/app-browser-image-downloader.git
   ```

2. 打开 Chrome 浏览器，访问 `chrome://extensions/`

3. 开启右上角的「开发者模式」

4. 点击「加载已解压的扩展程序」

5. 选择本项目文件夹

## 🚀 使用指南 | Usage

1. **打开扩展**
   - 访问任意包含图片的网页
   - 点击浏览器工具栏中的扩展图标

2. **选择图片**
   - 点击图片卡片选中/取消
   - 使用「全选」快速选择所有图片
   - 使用「仅大图」筛选高质量图片

3. **批量下载**
   - 点击「下载选中」按钮
   - 图片将自动保存到浏览器默认下载目录

4. **其他操作**
   - 「清除选择」- 取消所有已选图片
   - 查看图片尺寸信息
