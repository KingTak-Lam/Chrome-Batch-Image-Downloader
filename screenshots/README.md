# 截图说明

本目录存放扩展的宣传截图，用于上传到浏览器扩展商店。

## 截图要求

### Chrome Web Store
- **尺寸**: 1280x800 或 640x400 像素
- **格式**: PNG 或 JPEG
- **数量**: 至少1张，建议3-5张
- **内容**: 展示扩展的主要功能和界面

### 建议的截图内容

1. **主界面截图** (screenshot-main.png)
   - 展示扩展的主界面
   - 显示图片网格和控制按钮
   - 显示图片计数和选择状态

2. **选择功能** (screenshot-selection.png)
   - 展示图片被选中的状态
   - 显示不同的选择方式（全选、大图筛选等）

3. **下载功能** (screenshot-download.png)
   - 展示下载过程中的状态提示
   - 显示下载成功的提示

## 如何创建截图

1. 在 Chrome 浏览器中加载扩展
2. 访问一个包含多张图片的网页（如电商网站、图片网站）
3. 点击扩展图标打开弹窗
4. 使用截图工具捕获界面
5. 调整截图尺寸为 1280x800
6. 保存到此目录
