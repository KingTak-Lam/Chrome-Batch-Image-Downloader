# 隐私政策 | Privacy Policy

**最后更新日期**: 2026年1月19日  
**Last Updated**: January 19, 2026

## 中文版

### 概述

批量图片下载器（以下简称"本扩展"）尊重并保护用户隐私。本隐私政策说明了本扩展如何收集、使用和保护您的信息。

### 数据收集

**本扩展不收集任何个人数据。**

具体而言：
- ❌ 不收集浏览历史
- ❌ 不收集个人身份信息
- ❌ 不收集下载记录
- ❌ 不使用 Cookie 或类似跟踪技术
- ❌ 不向任何第三方发送数据

### 权限说明

本扩展请求以下权限，仅用于其核心功能：

1. **downloads** - 用于将图片保存到您的设备
2. **activeTab** - 用于读取当前标签页中的图片元素
3. **scripting** - 用于在页面中执行图片扫描脚本
4. **host_permissions ("<all_urls>")** - 用于支持跨域图片下载

### 数据处理

所有数据处理均在本地完成：
- 图片扫描在您的浏览器中进行
- 图片下载直接从源网站到您的设备
- 不经过任何中间服务器

### 数据存储

本扩展不在本地或云端存储任何数据。每次打开扩展时，数据都是实时从当前网页读取的。

### 第三方服务

本扩展不使用任何第三方分析、广告或跟踪服务。

### 儿童隐私

本扩展不针对13岁以下儿童，也不会故意收集儿童的个人信息。

### 隐私政策变更

如果隐私政策发生重大变更，我们会在扩展更新说明中通知用户。

### 联系我们

如有任何隐私相关问题，请通过 GitHub Issues 联系我们：  
https://github.com/airobus/app-browser-image-downloader/issues

## English Version

### Overview

Batch Image Downloader (the "Extension") respects and protects user privacy. This privacy policy explains how the Extension collects, uses, and protects your information.

### Data Collection

**This Extension does NOT collect any personal data.**

Specifically:
- ❌ No browsing history collected
- ❌ No personally identifiable information collected
- ❌ No download records collected
- ❌ No cookies or similar tracking technologies used
- ❌ No data sent to any third parties

### Permission Explanation

This Extension requests the following permissions solely for its core functionality:

1. **downloads** - To save images to your device
2. **activeTab** - To read image elements from the current tab
3. **scripting** - To execute image scanning scripts on the page
4. **host_permissions ("<all_urls>")** - To support cross-origin image downloads

### Data Processing

All data processing is done locally:
- Image scanning occurs in your browser
- Image downloads go directly from source websites to your device
- No intermediate servers involved

### Data Storage

This Extension does not store any data locally or in the cloud. Each time you open the Extension, data is read in real-time from the current webpage.

### Third-Party Services

This Extension does not use any third-party analytics, advertising, or tracking services.

### Children's Privacy

This Extension is not directed at children under 13 and does not knowingly collect personal information from children.

### Privacy Policy Changes

If there are significant changes to the privacy policy, we will notify users in the extension update notes.

### Contact Us

For any privacy-related questions, please contact us via GitHub Issues:  
https://github.com/airobus/app-browser-image-downloader/issues
